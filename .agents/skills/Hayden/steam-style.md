# Steam Style Guidelines

When working on the frontend of this project, adhere strictly to the following Steam-like design principles:

## Color Palette
*   **Primary Backgrounds**: Deep dark blues and greys. Use `#171a21` for the main background and `#1b2838` for secondary section backgrounds (like containers or navigation bars).
*   **Accents**: Use Steam's signature light blue `#66c0f4` for links, active states, and primary styling emphasis.
*   **Buttons**:
    *   Primary Calls to Action (e.g., "Play Game", "Buy"): Use a vibrant green gradient (e.g., `linear-gradient(to right, #75b022, #588a1b)`) with white text.
    *   Secondary Buttons: Use translucent dark blues (e.g., `rgba(103, 193, 245, 0.2)`) with the accent blue text.
*   **Text**: Use `#c6d4df` or `#acb2b8` for primary body text and `#8f98a0` for secondary/muted text. Use pure white `#ffffff` primarily for prominent headings, active states, or hover text.

## Typography
*   **Fonts**: Rely on clean sans-serif fonts like `Motiva Sans` (if available), `Arial`, `Helvetica`, or system UI fonts.
*   **Structure**: Ensure clear visual hierarchy using distinct font sizes and weights.

## Layout & Components
*   **Navigation**: A prominent, dark, horizontal navigation bar at the top with clean, easily readable categories.
*   **Cards/Capsules**:
    *   Use distinct rectangular "capsules" (images mapping to games/items).
    *   Interactive hover effects are crucial: caps should subtly brighten, display a border, or reveal more information on hover.
    *   Use dark, semi-transparent overlays for any text resting directly on images.
*   **Pricing/Tags**: Display distinct pricing tags (often with a dark background or Steam green for discounts) and small, discrete genre tags.

## Vibe & Atmosphere
*   The style should feel premium, heavily dark-themed, data-rich but neatly organized, placing the strongest visual emphasis on high-quality images and media over empty white space.
